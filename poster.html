üìã How to Arrange Your Poster Elements

üéØ Key Principles:

  • Visual Hierarchy: Largest element = Main offer ("FREE 1 DAY STAY")
  • Eye Flow: Top to bottom, left to right (Z-pattern)
  • White Space: Don't overcrowd - leave breathing room
  • Color Balance: Use 2-3 main colors + accent (gold for "FREE")

üìê Element Placement Guide:

  • Top 30-40%: Hero image/main offer with "1 DAY FREE STAY" text
  • Middle 40-50%: Icons, benefits, and step-by-step process
  • Bottom 10-20%: QR code, CTA button, and logo

üñºÔ∏è Icon Arrangement:

  • Grid Layout: 2x2 or 4 in a row for benefits (hotel, restaurant, time, free)
  • Circular Flow: Around the main text for modern look
  • Left Sidebar: Vertical stack with text descriptions
  • Size: 40-60px for feature icons, 80-100px for hero icon

üì± QR Code Placement:

  • Best Position: Bottom center or bottom right
  • Size: At least 2x2 inches (5x5 cm) for easy scanning
  • Surrounding: Add "SCAN HERE" text and arrow pointing to it
  • Padding: Keep 0.5 inch clear space around QR code

‚úçÔ∏è Text Guidelines:

  • "FREE": Make it 2-3x larger than other text, use gold/yellow
  • Headline: 60-80pt bold font
  • Subheading: 24-36pt regular/medium
  • Body Text: 14-18pt, max 8-10 words per line
  • Steps: Numbered 1-4, use icons + short phrases

üé® Tools to Use:

  • Canva: Use poster templates (18x24 or A3 size)
  • Figma: For precise design control
  • Photoshop: For advanced image editing
  • PowerPoint: Quick mockup with shapes and text
üí° Pro Tip: Print a test version at actual size before mass printing. Check if QR code scans properly and text is readable from 3-5 feet away. Use high-resolution images (300 DPI for print, 150 DPI for digital displays).
‚ö†Ô∏è Common Mistakes to Avoid:
  • Making QR code too small (minimum 2x2 inches)
  • Using too many fonts (stick to 2-3 max)
  • Overcrowding with too many icons
  • Low contrast text on busy backgrounds
  • Forgetting contact info or website URL
